A New Level WYSIWYG Editor

All it starts with a 1 File!

Why to make things bulky with lots of JS,CSS.. files! Adex Editor itself works with a single JS file and which further creates the respective styles and html for the user. All you need to do is just!



$(elem).adexeditor();

Inbuilt Image upload Plugin!

Adex Editor comes with image uploading plugin! All you need to do is, Tell the path of upload file to the plugin. It is also supports the uploading images from url and also varifies if there is image is not!



$(elem).adexeditor({ uploadpath: 'upload.php' });

Tables? Just one click away!

Creating the html tables would never be so easier! Just put the number of rows and columns and see theres is a table! The tables are also easy to edit just click on it and edit it as you need!

Custom Type Faces!

Caligraphy speaks! Make you content more speakable with the custom type faces! All yo need is a click over your favourite type face!

Infinte Colors!

Select the best of your color! There are limitless colors waiting for you with adex editor! Put the color to highlight the content or just to change the font color!

And much more!

The features aint finished here! It has lots of more features like custom code editing, adding link with optional new target attribute, creating ordered or unorderd list and much more..
